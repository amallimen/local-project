# Image Directory

Place your images here:

## Required Images:

1. **default.jpg** - This image will be displayed by default on the welcome page
   - Path: `public/assets/img/default.jpg`
   - Recommended size: 500x500px or similar aspect ratio
   - Formats: JPG, PNG, or GIF

2. **krishna.jpg** - This image will be shown when the "To See Krishna Click Here" button is clicked
   - Path: `public/assets/img/krishna.jpg`
   - Recommended size: 500x500px or similar aspect ratio
   - Formats: JPG, PNG, or GIF

## How to Add Images:

1. Copy your default image and name it `default.jpg`
2. Copy your Krishna image and name it `krishna.jpg`
3. Place both files in this directory: `public/assets/img/`

The page will automatically detect and display these images. If images are not found, a placeholder message will be shown.
